# Black Crown Barber

Landing page moderna para uma barbearia premium fictícia, desenvolvida com HTML, CSS e JavaScript. O projeto foi criado com foco em design sofisticado, experiência do usuário e responsividade, inspirado em marcas de luxo e barbearias de alto padrão.

## Sobre o Projeto

A Black Crown Barber é uma landing page que apresenta uma identidade visual elegante em tons de preto e dourado, destacando os principais serviços da barbearia, sua história, galeria de trabalhos, depoimentos de clientes e chamadas para ação voltadas para agendamentos.

## Funcionalidades

- Layout responsivo para desktop, tablet e dispositivos móveis
- Hero section com destaque visual e imagem principal
- Menu de navegação adaptável para telas menores
- Seção de serviços
- Galeria de imagens
- Seção institucional
- Depoimentos de clientes
- Call to Action (CTA)
- Footer com informações de contato
- Ícones utilizando Lucide Icons

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Google Fonts
- Lucide Icons
